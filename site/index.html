Why

You have agents now. You don't have a place to put them. Tickets pile up in Linear; the agent runs in your terminal; the work product ends up in a gist. The conductor is invisible — and the conductor is you.

VibeMaestro is the dashboard for this new shape of work. The board is the product. The agent is the worker. You stay the conductor.

How

Drop a one-liner. The local Claude Code or Codex CLI picks it up — running in a real PTY in a real shell with your real PATH. Bytes stream into the card; the state machine moves Backlog → Running → Reviewing → Complete on the agent's exit code. Approve or request changes. The board never lies.

What

  • Four-lane board, terminal-dark theme, color-blind-safe status.
  • xterm.js panel shows the live PTY, replay-on-attach to recently-ended runs.
  • Single-binary install. Local SQLite. No cloud.
  • MIT license, every commit reviewable.
